## Plan: 基于 RCWL-1605 超声波的定高控制系统

**TL;DR** 
利用现有的向下指的超声波测距仪作为高度反馈（仿地飞行）。采用高度-垂直速度的双级联 PID（外环求速度，内环求推力），外加 25% 的前馈悬停油门实现平稳悬停。定高模式将由独立编写的 MAVLink 的消息指令进行激活和退出。

**Steps**
1. **传感器接入与状态观测 (并行)**
   - 在主循环中轮询 RCWL-1605。在高度状态机中新增对地绝对高度 `alt_est` (m) 和 垂直运动速度 `vz_est` (m/s) 两个变量。其中速度是通过对超声波位移差一阶低通滤波求导取得。
2. **MAVLink 指令解析与模式切换 (串行，依赖 MAVLink 通信)**
   - 新增 MAVLink 协议解析（如 `MAV_CMD_DO_SET_MODE` 或某个自定义 MAV_CMD）。
   - 定义一个开关标记量（如 `alt_hold_enabled`）。当收到开启指令时，立刻锁存当前的 `alt_est` 也就是目标定高点 `alt_target`。
3. **双环高度 PID 闭环控制 (串行，依赖步骤 1,2)**
   - **位置外环 `alt_pid`**：输入 高度误差 `alt_target - alt_est` -> 输出 目标爬升/下降速度 `vz_target` (并限幅于 ±0.5m/s 左右策源平滑)。
   - **速度内环 `vel_z_pid`**：输入 速度误差 `vz_target - vz_est` -> 输出 油门补偿增量 `throttle_adj`。
   - **合成总推力**：当 `alt_hold_enabled` 为真时，下发给电机的指令将接管为：`thrustTarget = 0.25f (前馈悬停油门) + throttle_adj`。
4. **参数录入与暴露地站调参 (串行，依赖步骤 3)**
   - 将高度环的内外环 P、I、D 相关参数，以及基础前馈悬停油门 `hover_throttle = 0.25` 等注册为全局参数，以便您可以不反复刷固件直接通过地面站图形化调参界面边飞边看波形调节。

**Relevant files**
- `src/Flix_Main.cpp` — 初始化与主循环插入超声波传感器的 read 调用
- `src/estimate.cpp` — 插入高度和垂直速度的状态观测过滤与求导微分逻辑
- `src/mavlink.cpp` — 增加自定义消息解析切入/退出定高模式逻辑
- `src/control.cpp` — 高度级联定高控制逻辑的核心注入，接管 `thrustTarget` 纯手动逻辑
- `src/parameters.cpp` 与 `src/pid.h` — 相关两层 PID 的实例注册及增广定高相关参数供地面站读写

**Verification**
1. **数据验效**：通过 MAVLink 将 `alt_est` 或 `vz_est` 透传给您的 QGroundControl 或自写上位机，观察数值在 0.03 ~ 4.5m 的范围内是否呈无跳变的平滑曲线。
2. **逻辑验效**：发送 MAVLink 触发指令；将飞机提起来时察觉旋翼减速，将飞机按下时察觉旋翼突然奋力加速补油门。
3. **闭环调参**：速度内环首调 P（拉低观察响应速度防震荡），高度外环再调 P，将前馈死区锁在 25% 使其实现半空静态完美悬停。

**Decisions**
- **基础油门前馈模型**：通过硬编码悬停基准值为 25%（可作为预设参数调整），这可以极大抵住重力的影响，让 PID 只负责微调，能获得较好抗风与反应速度反馈。
- **仿地飞行特性（Terrain Following）**：由于超声波所测高度全部为“绝对离地距离（AGL）”。飞机在飞越桌面、箱子或者台阶时会产生自然爬升跨越并保持固定相对重心的特性。
